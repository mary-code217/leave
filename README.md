# 휴가 전자결재 시스템

사내 휴가 신청부터 승인까지의 전 과정을 관리하는 백엔드 API 서버입니다.

## 프로젝트 소개

기업 내 휴가 관리 업무를 디지털화한 시스템입니다.
직원은 휴가를 신청하고, 결재권자는 이를 승인/반려할 수 있습니다.

### 해결하고자 하는 문제
- 수기/엑셀 기반 휴가 관리의 비효율성
- 휴가 잔여일수 추적의 어려움
- 결재 프로세스의 불투명성
- 인수인계 누락으로 인한 업무 공백

## 기술 스택

| 구분 | 기술 |
|------|------|
| Language | Java 21 |
| Framework | Spring Boot 3.5 |
| Database | MySQL 8.0 |
| Auth | JWT (jjwt 0.12) |
| Build | Gradle |
| Docs | Swagger (springdoc-openapi 2.8) |

### 주요 라이브러리
- Spring Data JPA - 데이터 액세스
- Spring Security - 인증/인가
- Lombok - 보일러플레이트 코드 제거
- Bean Validation - 요청 데이터 검증

## 주요 기능

### 휴가 신청/결재
- 휴가 신청 (연차, 반차, 병가, 경조사 등)
- 다단계 결재 (승인 / 반려)
- 첨부파일 업로드 (진단서 등)
- 휴가 유형별 정책 설정
- 동시 휴가 인원 제한 정책

### 휴가 계정
- 입사일 기준 연차 자동 부여
- 휴가 사용 시 자동 차감
- 잔여일수 실시간 조회
- 사용 내역 원장(Ledger) 관리

### 인수인계
- 휴가 전 업무 인수인계 작성
- 인수자 지정 및 확인 처리

### 조직 관리
- 부서(Team) 관리
- 직급(Grade) / 직책(Position) 관리
- 사용자 계정 관리

### 기타
- 공휴일 자동 연동 (공공데이터포털 API)
- 알림 기능
- 감사 로그 (누가, 언제, 무엇을 변경했는지 추적)
